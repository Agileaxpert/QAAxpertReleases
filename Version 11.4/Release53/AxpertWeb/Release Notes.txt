    1. Based on the value of the 'isAxPut' flag, the Consumer Service determines how transaction data is saved to the database. If the flag is set to true, the Consumer Service uses the AxPut microservice to save the transaction data. Otherwise, it uses the ASBTStructRest.dll SaveData API. This applies to transaction data that is pushed to the CachedSaveQueue from Customized HTML pages or from forms configured for CachedSave.
